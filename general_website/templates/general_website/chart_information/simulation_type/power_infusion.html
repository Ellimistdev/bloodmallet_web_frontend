Power Infusion

This chart shows how much power each spec gains from getting access to one Power Infusion cast by a Priest in their group.

ATTENTION: APL support for Power Infusion is not wide spread. Please submit improvements to the Action Priority List (APL) in SimulationCraft.

It uses the "tier appropriate" SimulationCraft profile. 

# SimulatonCraft example override
## External Power Infusion
external_buffs.pool=power_infusion:120:1

## No external Power Infusion
external_buffs.pool=power_infusion:120:0

## Example APL line
actions+=/invoke_external_buff,name=power_infusion,if=buff.dancing_rune_weapon.up|!talent.dancing_rune_weapon

See the SimulationCraft Wiki for more information on how to set up your spec support.

When to use
Spec Explanation Last explanation update
Unholy Death Knight Use Power Infusion while Gargoyle is up, as well as Apocalypse or with Dark Transformation if Apocalypse or Gargoyle are not talented 2024-08-25 #6cd7f7ea
Vengeance Demon Hunter APL support exists but an explanation text is missing. 2024-08-25 #6cd7f7ea
Shadow Priest Use Power Infusion while Voidform or Dark Ascension is active. Chain directly after your own Power Infusion. 2024-08-25 #6cd7f7ea
Elemental Shaman Use Power Infusion on Cooldown. 2024-08-25 #6cd7f7ea
Enhancement Shaman APL support exists but an explanation text is missing. 2024-08-25 #6cd7f7ea
